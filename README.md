# KARAS Foundation Website

**Knowledge for Academic Recognition & Advancement Society**

A production-ready static website for KARAS Foundation, an NGO dedicated to empowering individuals and communities through education, skill development, and sustainable initiatives in Vizianagaram, Andhra Pradesh.

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Bootstrap 5.3
- AOS (Animate on Scroll) Library
- Font Awesome 6
- Google Fonts (Poppins + Playfair Display)

## Pages

1. **Home** - Hero, About Preview, Vision & Mission, Focus Areas, Why KARAS, Counters, CTA
2. **About** - Story, Vision/Mission/Philosophy, Purpose, Timeline, Core Values, Guiding Principles
3. **Programs** - Program Cards, 14 Core Objectives, Major Interventions, Working Districts
4. **Governance** - Board Members, Structure, Organogram, Beliefs, Policies
5. **Contact** - Info Cards, Contact Form, Google Map, FAQ, Newsletter

## Features

- Fully responsive (mobile-first)
- SEO optimized (meta tags, Schema.org, sitemap, robots.txt)
- Accessible (ARIA labels, semantic HTML, keyboard navigation)
- Animated (AOS, counter animations, parallax)
- Fast loading (lazy loading images, deferred scripts)
- Scroll progress bar
- Back to top button
- Sticky navigation with scroll effect

## Deployment

This is a static website ready for deployment on:
- **Vercel** - Drag and drop the folder
- **Netlify** - Connect to Git or drag and drop
- **GitHub Pages** - Push to repository and enable Pages

## Setup

1. Clone or download this repository
2. Place your logo file as `assets/logo/karas-logo.png`
3. Add a favicon as `assets/favicon/favicon.png`
4. Update the Web3Forms API key in `contact.html`
5. Deploy to your hosting platform
